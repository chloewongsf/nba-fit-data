NBA Fit Data Repository

Static JSON data for the NBA Fit project. This repository contains NBA player statistics and game logs for the 2024-25 season, served via GitHub Pages for the NBA Fit application.

Available Data

  • active_players.json
    Complete list of active NBA players for the 2024-25 season
  • players/ directory
    Individual player data including game logs, biographical information, and season statistics

Data Structure

Each player has three JSON files:

  • {player_id}_gamelog.json - Game-by-game statistics
  • {player_id}_info.json - Player biographical information
  • {player_id}_season_stats.json - Calculated season averages

This data is automatically generated from the NBA API and updated regularly. For more information about the NBA Fit project, visit the main repository.
